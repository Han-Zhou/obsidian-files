
### Rings 

#### Definition
A ring R is a non-empty set s.t. the axions:
- A1
- A2
- A3
- A4
- M2
- M3
- D1
holds.
[[Axioms and Integers]]

If, in addition, R has axiom (M1), we say that R is a <u>commutative ring</u> .

### Order Axioms

We have an order < on $\mathbb{Z}$, $\mathbb{Q}$, $\mathbb{R}$

s.t.
1. $$\begin{gather} \forall \ a, b \in \mathbb{Z} (\mathbb{Q}, \mathbb{R}) \\ exactly \ one \ of \ \\ a<b,\ a=b,\ a>b \ holds \end{gather}$$
2. $$\begin{gather} \forall \ a, b, c\in \mathbb{Z} (\mathbb{Q}, \mathbb{R}) \\ a>b \implies  a+c=b+c \end{gather}$$
3. $$ \begin{gather} If \ a>b \ \& \ c > 0 \implies a \cdot c > b \cdot c \\ If \ a>b \ \& \ c<0 \implies a \cdot c < b \cdot c \end{gather}$$
4. (Transitivity) $$\begin{gather} \forall \ a, b, c \in \mathbb{Z} (\mathbb{Q}, \mathbb{R}) \\ a<b \ \wedge b<c \implies a<c \end{gather}$$

In general, a commutative ring R with an order < s.t. axioms 1-3 hold is called a <u>ordered ring</u>.
